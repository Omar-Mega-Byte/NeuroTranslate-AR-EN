# NeuroTranslate-AR-EN

A Transformer-based neural machine translation system for Arabic to English, trained on parallel text data and designed to capture contextual and semantic relationships using attention mechanisms. This project implements a Seq2Seq Transformer model for Arabic→English translation.


## Installation

1. Clone the repository
2. Create a virtual environment:
   ```bash
   python -m venv nmt_env
   source nmt_env/bin/activate  # On Windows: nmt_env\Scripts\activate
   ```
3. Install dependencies:
   ```bash
   pip install -r requirments.txt
   ```
